Thanks to an update by Nicolas Nytko (CS PhD), Jupyter Lab workspaces now support collaborative editing by multiple users in group assessments. Students can now edit the same notebook and run code cells, seeing these changes occur in real time.

To take advantage of this feature, enable the group work feature on an assessment that is using the existing Jupyter workspace. If you already have group work enabled, then there is nothing extra to do!
