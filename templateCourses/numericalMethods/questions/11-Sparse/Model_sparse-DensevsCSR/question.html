Consider a ${{params.a}}\times{{params.a}}$ matrix of integers. There are $n$ nonzero entries, where $n\ll{{params.b}}$ and the rest of the entries are all $0$'s.

This matrix can be stored in two formats

1) Dense Representation: Each entry uses $4$ bytes of storage.

2) CSR format: Each column index, data value and row start pointer uses $4$ bytes of storage.

What is the largest value of $n$ (i.e. the largest number of nonzeros) for which storing in CSR format is more memory efficient?
